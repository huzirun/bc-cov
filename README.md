# SringBoot+Vue脚手架

## 介绍

这是一个极其简单的前后端分离项目脚手架

前端：Vue，Vue-Router，Vuex，Axios，ElementUI

后台：SpringBoot，Mybatis-Plus（不用写sql，简单！）

## 如何使用

后台：maven导入springboot工程，然后配置数据库，启动后台

前台：进入vue文件夹，执行命令：
```
npm install
```

```
npm run serve
```
即可启动前台vue工程
